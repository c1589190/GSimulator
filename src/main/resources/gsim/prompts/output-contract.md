# 输出合约

## 格式要求
- 使用 Markdown 格式。
- 关键人名、地名、势力名使用 **粗体**。
- 严格区分：
  - **Facts**: 基于数据文件明确记录的事实。
  - **Inferences**: 基于逻辑推断的结论。
  - **Hypotheses**: 无法确认的猜测。

## 引用要求
- 引用来源必须给出文件路径（如 data/worlds/default/entities.md）。
- Wiki 查询结果引用 import/web/prts.wiki/ 下的文件路径。

## 禁止事项
- 不要凭空编造实体名称和事件。
- 不要覆盖基础文件。
- 不要对不确定的事情下定论。
