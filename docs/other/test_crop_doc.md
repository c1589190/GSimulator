# 三国机密情报

## 第一章：董卓的阴谋

董卓率领西凉铁骑入京，控制了洛阳城。他废黜少帝刘辩，改立陈留王刘协为帝，是为汉献帝。

董卓与吕布密谋，计划在郿坞建造一座坚不可摧的堡垒，将朝廷的财富全部转移过去。

## 第二章：王允的计策

司徒王允对董卓的暴行深恶痛绝。他暗中联络了曹操、袁绍等忠义之士，准备除掉董卓。

王允府中藏有一把七星宝刀，锋利无比，削铁如泥。他计划将此刀交给一位勇士，假意献给董卓，实则伺机刺杀。

## 第三章：曹操的行动

曹操主动请缨，愿意携带七星刀前往董卓府中行刺。王允将宝刀交予曹操，叮嘱他务必小心。

曹操潜入董卓卧室，正要下手时，董卓突然翻身醒来。吕布也在门外守卫。曹操急中生智，假装是来献刀的。

## 第四章：吕布的背叛

吕布与董卓的侍妾貂蝉有私情。王允利用这一点，设下连环计，让吕布与董卓反目成仇。

吕布最终在凤仪亭亲手杀死了董卓，但天下并未因此太平。

## 第五章：后续发展

董卓死后，其部将李傕、郭汜率兵攻入长安，杀死王允，劫持了汉献帝。天下陷入更大的混乱之中。

曹操趁机崛起，收编青州黄巾军，势力迅速壮大，为日后的霸业奠定了基础。
